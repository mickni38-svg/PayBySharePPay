# paybysharepay AI Context — Ready To Pay Flow

## ReadyToPay
Når alle deltagere er Authorized/Reserved/Paid:
- GroupPayment status = ReadyToPay

## Overview
Overview skal vise:
- alle participant orders
- alle ordrelinjer
- hvem der mangler

## Betal nu
Kun host må kunne klikke:
“Betal nu”

Når host klikker:
- GroupPayment status = Completed/Paid
- ordre simuleres som frigivet til køkken
