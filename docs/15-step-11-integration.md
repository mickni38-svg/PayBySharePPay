# Step 11 – Integration

## Formål
At koble frontend og backend sammen.

## Krav der skal implementeres i dette step
- Kobl Angular services til API endpoints
- Implementér models/interfaces til API-svar
- Understøt flows som:
  - deltagersøgning
  - opret person/merchant
  - venner
  - opret ordre
  - hent ordreoverblik
  - registrer betaling
  - beskeder
- Ret nødvendige mapping- eller kontraktfejl

## Vigtige regler
- lav kun integrationsændringer
- undgå at introducere nye features
