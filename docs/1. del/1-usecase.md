# Use cases og acceptance criteria

## Use case 1 – Søg efter "venner"
Som bruger vil jeg kunne søge efter personer og merchants, så jeg kan vælge hvilke brugere der skal tilføjes til min venneliste.


### Acceptance criteria
- Vis alle brugere  og merchants som er oprettet som brugere af PayBySharePay
- deltager-type vises tydeligt
- søgning kan filtrere på navn eller relevant info
- Når man klikker på "Tilføj person til Vennelisten" så tilføjes de valgte deltagere til en personlig venneliste.
- Den personlige venneliste gemmes i databasen.
- De deltagere der er valgt som "Venner" skal ikke kunne fremsøges da de allerede er tilføjet til vennelisten.
