# Use cases og acceptance criteria

## Use case 1 – Søg efter "venner"
Som bruger vil jeg kunne søge efter personer og merchants, så jeg kan vælge hvilke brugere der skal tilføjes til min venneliste.


### Acceptance criteria

- Der skal være stor kontrast så det er tydeligt at se forskel på de forskellige elementer
- Black theme og hvid font
- Checked ring må gerne være hvid man kan næsten ikke se den
- Proffesionelt og enkelt design
