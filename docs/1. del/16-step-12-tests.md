# Step 12 – Tests

## Formål
At verificere centrale use cases og forretningsregler.

## Krav der skal implementeres i dette step
- Opret testprojekt
- Opret tests for services
- Dæk use cases som:
  - opret merchant
  - søg deltagere
  - opret ordre
  - registrer betaling
- Dæk centrale valideringsregler

## Vigtige regler
- fokuser på service-laget
- test forretningslogik, ikke styling
