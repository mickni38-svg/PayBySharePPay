# Koderegler – PayBySharePay

## Generelle regler

- vis hele filer, ikke kun kodeudsnit
- implementér kun det aktuelle step
- spring ikke videre til næste step
- brug tydelige namespaces
- brug meningsfulde klassenavne

## Backend

- .NET 9
- brug async/await
- brug dependency injection
- brug interfaces
- undgå forretningslogik i controllers
- undgå databaseadgang direkte i service-laget uden repository

## Frontend

- Angular + TypeScript
- brug services til HTTP
- brug typed models/interfaces
- hold komponenter overskuelige
- lav mobilvenlige styles

## EF Core

- konfigurer relationer tydeligt
- brug SQL Server provider
- gør modellen migrations-klar

## Dokumentation

I steps med dokumentation skal Copilot opdatere README eller docs-filer.
